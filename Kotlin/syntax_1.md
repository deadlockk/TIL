## 변수 선언 var

```
  var num1 : Int = 21_0000_0000
  
  var num2 : Float = 3.12f // float 는 f를 붙여주어야 함
  
  var num3 = 3.14 //Double
  
  // 뒤에 입력되는 type을 판단해서 스스로 지정해준다
```

## Data Type

* Byte(8bit)

    -128 ~ 127

* Short(16bit)

    -32,768 ~ 32,767

* Int(32bit)

    –2,147,483,648 ~ 2,147,483,647

* Long(64bit)

-9223372036854775808 ~ 9223372036854775807

* Float(32bit)

소수점 7자리

* Double(64bit)

소수점 14자리
